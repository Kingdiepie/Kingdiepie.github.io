My Site and the Test

I decided to choose my own website for this project becuase I thought it'd be more interesting then doing s well known site like facebook. In addition, I am in the middle of rebuilding this website and I wanted to know what elements users liked and what they htought was missing.

As for the actual test I had 5 tasks.

  • The first task was to find my Twitter account. While I don't use on a normally anymore I want the links to my social media to be easy to find but not intrusive. The links to social media are on the footer of my website so I thought it would be interesting to see if users would look there.
  • Find my "Flying Koopa" game and download the .rar. This test was to see if users used the home page or menu to find projects, and to see if users could download htings easily
  • Find my GPA calculator program and download it. This is very simliar tot he last task but there is a search bar, and oyu must scroll down on the gome page to see calculator programs
  • Find my single-player pong game. This is part of my HTML5 Arcade that is not accesable without useing the menu. I also wanted them to try and figure out controls and see if they knew how to pause the game
  • THe last test was easy, just return to the home page. I wanted to see if they would use the menu or click my name
