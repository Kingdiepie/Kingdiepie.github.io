HTML Game Creation

A intro to makeing a html canvas.

Here's how you make a HTML 5 game without any frame work: We'll start with the basic components then work on makeing a tron game. Basic HTML knowlage is required and basic JS/programming knowlage is recommended.

the HTML: 

<head>
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/2.2.0/jquery.min.js"></script>
</head>
<body>
  <canvas id="canvas">
  <script language="javascript" src="main.js"></script>
</body>

Nothing fancy just a link to Jquery(In case you need it later), your JS, and your canvas. If you're unfamiliar with HTML or need a refresher try http://www.w3schools.com/html

var canvas = document.getElementById("canvas");
var ctx = canvas.getContext("2d");
canvas.width = 800;
canvas.height = 600;
var i = 0;

This should go at the beginning of your JS file, it gives you control of your canvas, you can now set the high and width and draw on your canvas with ctx.fillwhatevermethod() 

window.onload = function() {
    var main = function() {
      var now = Date.now();
      var delta = now - then;
      update(delta / 2000);
      render();
      then = now;
      requestAnimationFrame(main);
    };
    var w = window;
    requestAnimationFrame = w.requestAnimationFrame || w.webkitRequestAnimationFrame || w.msRequestAnimationFrame || w.mozRequestAnimationFrame;
    var then = Date.now();
    main();
  };

Next is your basic game loop, update is where you will change variables and there values very frame and render is where you will draw everything based of those varibles. You can also change your game speed by increasing or decreasing FPS. This should always be your last black of code. 

var rectX = 200;
var rectY = 200;
var rectSpeed = 128;
ctx.font = "30px Arial";

We're gonna add these global variables next. 

var update = function(m) {
  i++;
};
var render = function() {
  ctx.fillStyle = "#FFFFFF";
  ctx.fillRect(0,0,canvas.width,canvas.height);
  ctx.fillStyle = "#000000";
  ctx.fillText("Number of frames ago you loaded me: " + i,50,50);
  ctx.fillRect(300,300,rectX,rectY);
};

The above code is your render and update methods. This is the code that is run every frame. Things should now get draw on your canvas. Don't worry if it looks bad, it should. There should be a number on your screen that counts up every frame.

We new technically have a game, if your familiar with Javascript you can probably make your own after looking at the all the canvas fill methods.

If your still not sure on how to make a real game look at the code for my Tron HTML 5 Game (Not the Java one).
